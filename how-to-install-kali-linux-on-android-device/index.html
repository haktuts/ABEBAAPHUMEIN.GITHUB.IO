Install kali linux in android device.

 


1.Download linux deploy android appliation.


2.Configuring the linux deploy for kali linux by choosing "kali linux" in "distribution tab"

 

 

 

 

 

 

 

 


 

 

 

 

 

 

3.Choose the approprite architecture i.e."armhf"


4.Set up the mirror URL "http://http.kali.org/kali/"


5.Set your installation type and location on your Android device correctly.


6. Click on install button now download will started.


7.Once the installation is complete, you can have Linux Deploy automatically mount and load up your Kali Linux.
