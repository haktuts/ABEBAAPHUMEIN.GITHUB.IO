IP Address

 

 All computers connected to the internet have an unique number, usually called an “IP ADDRESS”. IP Address individually identifies every computer on the  Internet.

 An IP Address consists of four numbers, separated by “dots”. Range of every number in between  0 – 255. Example:-  210.211.180.20

 

Types of IP address:

 

1. IPv4

2. IPv6

3. Public IP

4. Private IP

5. Static IP

6. Dynamic IP

7. Internal IP

8. External IP

 

 

Steps to checks  windows private IP address:

 

1.Start

2.Run

3.CMD

4.Ipconfig

 

Steps to checks private IP address in linux :

 

1.Ctrl+alt+F1

2.Ifconfig

 

Steps to check public IP address:

 

1.Open browser (i.e opera,chrome,mozile,IE)

2.www.whatismyIP.com
