Download youtube full playlist via IDM.

 

  1. Download youtube multidownloader

  2. goto the channel tab

  3. provide name of youtube channel

  4. Press CTRL+A to select all

  5. Right click to download the file

  6. Youtube multi downloader will sync all the playlists download link in IDM and click on check all & press OK

  7. finish

 

Note: Set IDM as your default downloader in Youtube Multi Downloader.
