Run Multiple Window On Any Android Device.

 

 

Method 1

 

1.Download and install multitasking android appliaion form android market.

 

2.Now configure the application as per your requirement.

Multitasking is composed by the settings menu, the sidebar and the floating tools. To access the tools bar do a swipe gesture to the right.


The windows can be modified in various ways:
- they can be resized;
- they can moved anywhere on the screen;
- they can be used in fullscreen mode;
- they can be minimized into the notification bar;

 

3.Then use the multiscreen in your android device.

 

 

Method 2

 

1.Download and install Multi Window Manager (Phone)

 

2.Run the application and use the benifit of multiscreen.

 

 

Note 1:  ROOT REQUIRED for method 2

 

Note 2:  REQUIRES A SAMSUNG DEVICE AND ROM THAT SUPPORTS MULTI WINDOW for method 2.
