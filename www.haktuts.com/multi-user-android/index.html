How to set up Multi user in android Smartphone.

 

 

1.  Download and install switch me application.

 

2.  Set the profile name of active account by click on create profile.

 

3.  Now click on plus icon (+) at the top of windows.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

4.  Enter the new profile name.

 

5.  Now user can easily switch between both the account.

 

Example: profile created in step 2 is admin and  profile created in step 3 is sample . So, user can easily switch from admin to sample and vice verse. After setting dual profile user can easily start dual whatsapp and also make your device dual boot.
